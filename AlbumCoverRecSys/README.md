# AlbumCoverRecSys

앨범 커버 이미지를 활용한 음악추천 및 데이터 분석

# 	주제 선정 배경
- 음악 추천에는 여러 데이터가 활용되고 있지만 아직 앨범 커버를 활용한 사례는 없다.
-	앨범 커버 이미지를 이용하여 추천 시스템을 만들어보면 재밌는 결과가 나오지 않을까?
